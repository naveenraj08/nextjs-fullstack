## Sanity CMS + Next JS 15.2 + Shadcn/ui TailwindCSS

A beginner blog app to explore next.js concepts, with CRUD operations.

### Features of the application

- OAuth Authentication
- Search Posts
- User Profile Management
- User Comments
- Responsive Design
- Dark Mode
